Create the `crates/bundle-cli/` directory structure. This is a standalone Cargo project (no workspace).

**Cargo.toml:**
```toml
[package]
name = "edgeworks-bundle"
version = "0.1.0"
edition = "2021"
description = "Create, verify, and inspect offline update bundles for air-gapped edge devices"

[[bin]]
name = "edgeworks-bundle"
path = "src/main.rs"

[dependencies]
clap = { version = "4", features = ["derive"] }
serde = { version = "1", features = ["derive"] }
serde_json = "1"
chrono = { version = "0.4", features = ["serde"] }
sha2 = "0.10"
indicatif = "0.17"
thiserror = "2"
```

**src/main.rs:** Use clap derive API. Define:
- `Cli` struct with `#[command(name = "edgeworks-bundle")]`, version, about
- `--json` as a global flag: `#[arg(long, global = true)]`
- `Commands` enum with three variants:
  - `Create` — with `--image` (required String), `--output` (required PathBuf), `--notes` (optional String, default ""), `--target-device` (optional String, default "any")
  - `Verify` — positional `path` (PathBuf)
  - `Inspect` — positional `path` (PathBuf)
- In `main()`: parse args, match on command, call stub functions from each module that print "not yet implemented" and return `Ok(())`
- Declare modules: `mod create; mod verify; mod inspect; mod manifest; mod error;`

**Stub modules (create.rs, verify.rs, inspect.rs):**
Each file: one `pub fn run(...)` that returns `Result<(), error::BundleError>` and does `todo!("not yet implemented")`. Import `crate::error`.

**manifest.rs:** Stub with comment `// Manifest types — implemented in plan 02`. Add empty placeholder structs so the crate compiles:
```rust
use serde::{Deserialize, Serialize};

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct BundleManifest {
    pub schema_version: String,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct BundleImage {
    pub reference: String,
}
```

**error.rs:** Stub with a minimal `BundleError` enum using thiserror so the crate compiles:
```rust
use thiserror::Error;

#[derive(Error, Debug)]
pub enum BundleError {
    #[error("not yet implemented")]
    NotImplemented,

    #[error("IO error: {0}")]
    Io(#[from] std::io::Error),
}
```

The stub types/errors will be replaced with full implementations in plan 02. The goal here is just to get `cargo build` passing.
