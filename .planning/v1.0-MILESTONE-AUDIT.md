## Integration Fixes Applied

Issues found by integration checker, fixed in commit `5f20dd3`:

1. **NODE_IP env var** — Added `status.hostIP` fieldRef to observability-stack.yaml pod spec. Without this, the `kubeletstats` receiver couldn't connect to kubelet metrics.
2. **Dead BOOTC_VERSION** — Removed from versions.txt, Makefile, build.sh, and load-versions action. Was passed as `--build-arg` but no `ARG` in Containerfile consumed it.
3. **Stale K3s examples** — Updated `update-version/action.yml` description text from K3s to OpenTelemetry examples.
